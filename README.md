# tensorflow-learnning
* fc2在fc1的基础上增加了“断点续训”功能；
* fc3在fc2的基础上增加了应用程序，实现了输入手写数字图片输出识别结果；
* fc4在fc3的基础上增加了数据集生成程序，实现了把7万张手写数字图片制作成数据集和标签，程序可以用自制数据集和标签训练模型并输出手写数字图片的识别结果。 mnist_data_jpg解压放在目录下即可运行
